Untuk Laptop/PC



	






1. Teks Sublim
Aplikasi Sublime merupakan suatu aplikasi text editor dengan dinamis yang sangat berguna untuk menulis
 sejumlah kode serta mampu membuka berbagai macam file. Selain itu, Sublime Text juga mendukung berbagai 
 pemrograman bahasa seperti PHP, Javascript, Python, C++, C, C#, CSS, ASP, dan sebagainya.

2. Studio Kode Visual
Disesain tengan tampilan yang menarik, elegan dan dinamis. Text Editor ini mendukung banyak bahasa 
 pemrograman, dan juga mendukung fitur integrase git dan debugging, sehingga kita perlu mencari kesalahan
 Codingannya sendiri namun fitur tersebut akan memeriksa kesalahannya sendiri.

Bahasa Pemrograman yang didukung seperti C, C++, Java, Java Script, PHP dan Bahasa Pemrograman lainnya.

3. Notepad++
Notepad++ juga merupakan editor teks yang dikembangkan dari Notepad. Notepad++ sendiri memiliki kelebihan 
 dibandingkan dengan Notepad yang salah satunya adalah mendukung bahasa pemrograman. Bahasa 
 Pemrograman yang didukung diantaranya ada C++, Assembly, Python, Java Script, PHP, VB dan lainnya.
 
Text editor dapat kamu download secara gratis di website nya langsung, tanpa harus membayar uang sepeser 
pun.

4. Vim
Vim atau Vi Improved yang bisa dibilang text editor yang powerfull dan terpakai sampai sekarang. ini sudah
 lama dikenal dan diluncurkan pada tahun 1991, tetapi walaupun sudah lama, semakin meningkat penggunanya
 namun sekarang menjadi salah satu Text Editor terbaik dan banyak digunakan oleh Programmer.

5. Braket
Aplikasi Bracket memang hanya dikhususkna untuk Desainer Web. Jadi hanya bisa kamu gunakan Bahasa 
 Pemrograman atau yang berhubungan dengan Web saja seperti HTML, CSS dan Javascript.



Aplikasi ini dapat kamu unduh secara gratis pada Websitenya, dan untuk sistem operasinya juga tidak perlu 
 kamu khawatirkan, karena ia mendukung berbagai sistem operasi, seperti MacOs, Linux dan Windows.
